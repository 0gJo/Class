모든 html요소는 속성(attribute)을 가진다. 속성 - html 요소의 추가정보를 제공 속성은 시작태그 안에 선언 문법 : 시작태그 속성명="속성값" 속성명='속성값' 속성명=속성값 src ="url" alt="CDATA" 문자열 URL (Uniform Resource Locators) - 브라우저가 서버에 웹페이지를 요청할때 사용되는 주소(웹주소) - 구성 schema:// prefix.domain:port/path/filename schema:인터넷서비스 유형 (http,https,ftp,file) prefix : 도메인 접두사 (www , kr, jp ...) port 호스트의 포트번호 http의 기본포트 (80) path 서버의 경로 filename 문서 or 자원이름 [http] : // [www] .[ naver.com] URL 인코딩 - URL은 ASCII 문자 집합을 사용하여 인터넷을 통해서만 전송할 수 있다. - 지금처럼 비 ASCII 문자 집합(한글)은 변환되어진다 - %16진수값으로 대체됨. - URL은 공백을 포함할 수 없다. 공백은 + or %20으로 대체됨.
